KANDI - User Management System

KANDI UMS is a PHP application built up around the CodeIgniter framework(version 1.0.0), that allowing the registration and administration of users.It can be used as a base platform for any CodeIgniter based web applications. It provide secure login, authentication, authorization and complete user management.

If you already have an existing PHP application, and you want to add user login, registration and user role management features,then you need to move your application to CodeIgniter framework and incorporate it inside KANDI UMS application.

Server Requirements

  • MySQL (5.1+) via the mysqli

How to use

  1. Login form

    2. Enter your email and password and click log In button. Now you can see the dashboard.

  2. Dashboard

    3. In your dashboard you can see all information in simple way.

  3. Profile

    4. Here you can see list of Employees and can edit your profile details.

  4. User Management(search,delete,activate/deactivate)

    5. Add User
    To add a new user, click on the User Management-> Add user link. A View will open. On the left, you will Add user Form. On the right, you will see those list of users. Fill the fields and submit the form.You should fill all required fields.
    Delete User
    For delete user you have to select the user from the list and then click on the "Delete" button.
    Search User
    User table contain a search field at the top right of the table.You can search a user with username/email or phone.
    Activate/Deactivate User
    If a user is active, you will see Deactivate button otherwise Activate button .



  5. Company Profile

    6. Under the General settings->Profile Settings, You Can add your business/company profile, Like Business/Company name, address, contact details, company logo.

  6. Add Menus

    7. To add new menu, go to User management->Add menu link. View will open showing your list of menus and a form. Enter Menu name, Menu Url (controller/function), Order of the menu and its parent menu.




  7. Add Groups/Roles

    8. To add new group / role, go to General Settings->Add Group link. View will open showing your list of groups and a form. Enter Group name. To Give permissions to each group, click on Permissions button in-front of the group.



    Here you will see your menus with CRUD Permissions like View, Insert, Update, Delete. What it mean's is which user of a group can see which menus, the forms inside that menu, permission will be applied like Insert or Edit Permission. Instead of placing a Button in a form just place "$My_Controller->savePermission;" in PHP tags and if this user have insert permission for this form/ menu he will see the button otherwise he won't. For Edit ($My_Controller->editPermission;) For Delete ($My_Controller->deletePermission;)
